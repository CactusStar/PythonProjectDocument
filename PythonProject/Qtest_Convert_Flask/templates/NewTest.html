Test Process
# Procedure Verification Procedure
1

Procedure (1):

  • Open the testfile identified as K57_TW001 in the Setup testcase for this testcase and remain offline.
  • Verify the controller.  The controller should verify with 0 errors and 0 warnings.
  • This testfile has three (3) K5700 modules all pre-configured to be in Bus Sharing Group 1 and with Demo Mode configured as "Active".
  • On the Advanced Dialog (select the "Advanced…" Button) on the Power Tab of the Module Properties Dialog for the module named "Drive_K5700_DAI_BSG1_IP101", change the Demonstration Mode Selection from "Active" to "Inactive".
  • Apply the change.
  • Verify the controller.

Verification Procedure (1):

The verify will show 4 errors as follows:

  • Error: <axisname1>: Bus Sharing Group 1 contains modules configured with different Demonstration Mode selections.
  • Error: <axisname2>: Bus Sharing Group 1 contains modules configured with different Demonstration Mode selections.
  • Error: <axisname3>: Bus Sharing Group 1 contains modules configured with different Demonstration Mode selections.
  • Error: <axisname4>: Bus Sharing Group 1 contains modules configured with different Demonstration Mode selections.

The change to "Drive_K5700_DAI_BSG1_IP101" done in Procedure (1) caused the 3 K5700 modules in Bus Group 1 to (now) have differing Demonstration Mode Selections.  The verification process develops an error for each axis that points to a Bus Group 1 K5700 module.  Since the DAI module has 2 axis pointing to it, there are two identical error messages developed for the DAI module.

Leave the change in effect and move to the next procedure.
2

Procedure (2):

  • On the Advanced Dialog (select the "Advanced…" Button) on the Power Tab of the Module Properties Dialog for the module named "Drive_K5700_HPI_BSG1_IP041", change the Demonstration Mode Selection from "Active" to "Inactive".
  • Apply the change.
  • Verify the controller.

Verification Procedure (2):

The verify will show 4 errors as follows: (these are identical to the 4 errors seen in the previous Verification Procedure)

  • Error: <axisname1>: Bus Sharing Group 1 contains modules configured with different Demonstration Mode selections.
  • Error: <axisname2>: Bus Sharing Group 1 contains modules configured with different Demonstration Mode selections.
  • Error: <axisname3>: Bus Sharing Group 1 contains modules configured with different Demonstration Mode selections.
  • Error: <axisname4>: Bus Sharing Group 1 contains modules configured with different Demonstration Mode selections.

The change to "Drive_K5700_HPI_BSG1_IP041" done in Procedure (2) continues the condition where there are 3 K5700 modules in Bus Group 1 and they (still) have differing Demonstration Mode Selections.

Leave the changes in effect and move to the next procedure.
3

Procedure (3):

  • On the Advanced Dialog (select the "Advanced…" Button) on the Power Tab of the Module Properties Dialog for the module named "Drive_K5700_DFE_BSG1_IP031", change the Demonstration Mode Selection from "Active" to "Inactive".
  • Apply the change.
  • Verify the controller.

Verification Procedure (3):

The verify should show 0 errors and 0 warnings.

The change to "Drive_K5700_DFE_BSG1_IP031" done in Procedure (3) now results in the condition where there are 3 K5700 modules in Bus Group 1 and they all have the same Demonstration Mode Selection = Inactive.

This testcase verifies that all modules in a Bus Sharing Group must have the same Demo Mode Configuration (all either "Active" or all "Inactive".

Close the testfile without saving.
